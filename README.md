# AzureCycleCloudHPCproject
Este repositorio documenta los pasos, configuraciones y pruebas realizadas durante la implementación de un clúster HPC basado en Slurm, administrado mediante Azure CycleCloud. El objetivo es demostrar la comprensión de los conceptos principales para aprovisionar, monitorear, escalar y controlar costos en entornos HPC dentro de Azure.
